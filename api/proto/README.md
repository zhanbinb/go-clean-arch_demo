# Protocol Buffers

This directory contains the source `.proto` files. Generated Go code goes to `../gen/`.

## Layout

```
api/
├── proto/
│   ├── article/v1/article.proto
│   ├── author/v1/author.proto
│   └── auth/v1/auth.proto
└── gen/          # Generated by buf (gitignored)
```

## Generate

```bash
make proto
```
